# 🎧 Spotify Web Player Clone

A responsive frontend UI clone of the Spotify Web Player built using **HTML** and **CSS**.  
This project focuses on layout design, responsiveness, and interactive hover effects.

---

## 🚀 Features

- 🎨 Clean and modern Spotify-inspired UI  
- 📱 Fully responsive design (works on different screen sizes)  
- 🖱️ Smooth hover effects for better user experience  
- 📂 Structured layout similar to Spotify Web Player  
- ⚡ Built using only HTML & CSS (No JavaScript)

---

## 🛠️ Tech Stack

- HTML5  
- CSS3
